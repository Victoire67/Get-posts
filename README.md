# Fetch Posts

Fetch Posts is a web-based application that retrieves and displays blog posts from a specified API endpoint. Simply input the URL and press **Enter** or click **Fetch** to see the results.

## Tech Stack

- **React.js** (Frontend Library)
- **Tailwind CSS** (Styling)
- **TypeScript** (Type Safety)

## Getting Started

We welcome contributions! To get this project running on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Victoire67/Get-posts.git](https://github.com/Victoire67/Get-posts.git)
   ```
2. navigate to the directory

```bash
 cd Get-posts
```

3. install dependencies

```bash
npm install
```

4. Run the developement server

```bash
npm run dev
```

## Usage

Enter a valid JSON API endpoint (e.g., https://jsonplaceholder.typicode.com/posts) into the input field to fetch and display post data.
